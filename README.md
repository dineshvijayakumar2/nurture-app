## üöÄ Key Modules & Features

### 1. üè† Home (Dashboard)
The heart of Nurture is a comprehensive dashboard that provides a complete view of your child's developmental journey.

- **Calendar View**: Full monthly calendar showing all scheduled activities with visual indicators for busy days
- **Upcoming Classes**: Smart sidebar listing the next 7 days of scheduled activities with AI-generated icons
- **"Next Up" Hero Card**: Prominently displays the immediately upcoming activity with time and details
- **Activity Feed**: Beautifully designed masonry layout combining:
  - Recent journal entries with proper parent attribution
  - AI-generated developmental insights
  - Knowledge base highlights
- **Real-time AI Processing**: Live "Gemini analyzing..." indicator when generating insights
- **Premium Aesthetics**: Soft gradients, glassmorphism, and high-legibility typography

### 2. üìù Journal (Intelligent Logging)
More than just a journal, this is the primary data source for Nurture's intelligence.

- **Multi-Parent Support**: Each log properly attributes to the logged-in parent, not the child
- **Family Collaboration**: All family members can contribute observations and see who wrote what
- **Voice Recording**: Record observations hands-free with the microphone button‚ÄîGemini transcribes your audio to text with confirmation before saving
- **Multimodal Capture**: Seamlessly record text observations and voice memos
- **Intelligent Extraction**: Gemini automatically scans your logs to detect potential new activities (e.g., "Sarah started piano") and offers one-click scheduling into Activities
- **Real-time Sync**: Cross-device persistence powered by high-concurrency Firestore
- **Privacy Controls**: Mark entries as private for sensitive observations

### 3. üìä Activities (Smart Scheduling & Analytics)
A revolutionary take on family calendars, optimized for the complexity of child development.

- **AI-Generated Icons**: Each activity automatically gets a unique, watercolor-style icon generated by Gemini
- **Multi-State Views**: Toggle between high-level **Month** views, detailed **Week** columns, and a management-focused **Series List**
- **Advanced Scheduling**: Support for recurring classes, date ranges, start/end dates, and status management
- **Exception Dates**: Create one-time event overrides without duplicating recurring schedules (Google Calendar-style)
- **Flexible Editing**: Edit individual occurrences or entire series with granular control
- **Calendar-Based Stats**: View activity statistics by This Month, Last Month, This Year, or Last Year
- **Smart Drilldown**: Click any activity to see detailed history filtered by the selected time period
- **Bulk Date Helpers**: Unique tools to schedule 4-week workshops or weekend classes with a single tap
- **Cost Tracking**: Monitor investment in activities and classes

### 4. üå± Values (Value Garden & Wisdom Library)
A digital sanctuary for values-based parenting and resource management.

- **Value Garden**: Explore and nurture family values through AI-generated, contextual dialogues
- **Wisdom Library**: A dedicated repository for Books, Articles, and Videos. Manage a family-wide "Wisdom Stack" with live Add/Delete capabilities
- **Reader View**: A distraction-free, "Kindle-style" interface for deep reading on mobile devices
- **Contextual Dialogues**: AI-generated conversation starters that are **grounded in your child's actual history**. If a log mentions a child was brave at the dentist, the "Brave" value starter will specifically reference that moment
- **Knowledge Integration**: All AI responses incorporate your saved wisdom sources for personalized advice

### 5. ü§ñ Companion (The Contextual AI Coach)
Your dedicated parenting expert, powered by the most recent context of your family's life.

- **Full-Context Memory**: The Companion doesn't just answer questions; it knows your child's recent logs, schedules, and challenges
- **Persistent Conversations**: Chat histories are stored per-user and synced across devices
- **Multimodal Input**: Ask questions with text or photos
- **Markdown Support**: Richly formatted responses for clear, actionable advice
- **Knowledge-Aware**: Incorporates your saved research and wisdom sources

### 6. üë• Family Profiles (Multi-User Management)
Collaborative family environment with clear role management.

- **Separate Profiles**: Clear separation between child profile and parent profiles
- **Parent Management**: Each parent has their own profile with name, role (primary/partner/family), and photo
- **Child Profile**: Dedicated section for child's name, age, photo, temperament, activities, and dietary preferences
- **Family Sharing**: Share a single Family ID to collaborate across multiple parents/caregivers
- **Proper Attribution**: All journal entries and activities show which parent created them

## üß† Advanced AI Features Powered by Google Gemini

Nurture leverages **Google Gemini 3.0** across six critical AI operations:

### 1. **Intelligent Log Extraction** (`gemini-3-flash-preview`)
- Automatically extracts structured data from freeform journal entries
- Detects moods, activities, durations, and developmental observations
- Identifies schedule changes mentioned in observations (e.g., "Started Ballet on Mondays")
- Suggests one-click scheduling for mentioned activities

### 2. **Voice Transcription** (`gemini-3-pro-preview`)
- Converts audio recordings to text with high accuracy
- Hands-free observation capture for busy parents
- Confirmation modal allows editing before saving
- Supports multiple recording sessions that append to journal entry

### 3. **Neural Reading & Insights** (`gemini-3-pro-preview`)
- Analyzes weekly patterns to generate comprehensive developmental insights
- **Growth Architecture**: Poetic naming of developmental phases (e.g., "The Curious Connector")
- **Current Reading**: Pattern recognition across recent observations
- **Forecast**: Predictions of upcoming developmental shifts
- **Science Background**: Explains the developmental "why" behind observed patterns
- **Knowledge Integration**: Incorporates family wisdom sources into insights

### 4. **Activity Icon Generation** (`gemini-2.5-flash-image`)
- Automatically generates beautiful, watercolor-style icons for each activity
- Creates consistent visual language across the app
- Pure white background, whimsical aesthetic
- On-demand generation for new activities

### 5. **AI Companion** (`gemini-3-pro-preview`)
- Provides research-backed parenting advice grounded in your child's specific history
- Maintains context across multi-turn conversations
- Multimodal input support (text and images)
- Integrates your saved knowledge sources for personalized responses

### 6. **Value Dialogues** (`gemini-3-pro-preview`)
- Generates contextual conversation starters for teaching values
- Grounds advice in actual moments from your child's life
- Provides philosophical roots and practical teaching moments
- References family parenting philosophies from wisdom library

## üíª Tech Stack & Architecture

### Frontend
- **Framework**: React 19.2.3 with Vite 6.2 for lightning-fast builds
- **Type Safety**: TypeScript 5.8 for robust type checking
- **Routing**: React Router 7.13 for seamless navigation
- **State Management**: React Context API with custom FamilyProvider
- **Styling**: Tailwind CSS with custom design system (soft gradients, glassmorphism, premium aesthetics)

### Backend & Infrastructure
- **Authentication**: Firebase Auth with Google Sign-In
- **Database**: Firestore for real-time, low-latency document sync
- **Storage**: Base64 image encoding for photos (no separate storage bucket needed)
- **Hosting**: Firebase Hosting on global CDN
- **Data Model**: Family-based architecture (`families/{familyId}/collections`)

### AI Integration
- **AI SDK**: @google/genai v1.38.0 (Google Gemini SDK)
- **Models Used**:
  - `gemini-3-pro-preview`: Neural readings, AI companion, value dialogues, voice transcription
  - `gemini-3-flash-preview`: Intelligent log extraction with structured output
  - `gemini-2.5-flash-image`: Activity icon generation
- **Multimodal Capabilities**:
  - Text + image analysis for journal entries
  - Audio transcription for voice-based journal entries
  - Image generation for activity icons

### Mobile & PWA
- **Progressive Web App**: Full offline support with service workers
- **Capacitor.js**: Ready for native iOS and Android app conversion
- **Responsive Design**: Mobile-first design with desktop optimization

## üõ†Ô∏è Setup & Installation

### Prerequisites
- Node.js 18+ and npm
- Firebase account
- Google Gemini API key

### Local Development
```bash
# Clone the repository
git clone https://github.com/dineshvijayakumar2/nurture-app.git
cd nurture-app

# Install dependencies
npm install

# Set up environment variables
# Create a .env file with:
# API_KEY=your_gemini_api_key_here

# Start development server
npm run dev
```

### Firebase Setup
1. Create a Firebase project at [firebase.google.com](https://firebase.google.com)
2. Enable Authentication (Google Sign-In)
3. Create a Firestore database
4. Update `firebaseConfig.ts` with your project credentials

### Production Build & Deployment
```bash
# Generate optimized production bundle
npm run build

# Deploy to Firebase Hosting
npx firebase login
npx firebase deploy --only hosting
```
